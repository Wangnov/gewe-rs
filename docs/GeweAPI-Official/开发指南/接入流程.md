# 接入流程

:::highlight purple 
**步骤1：** 进入[用户主页](http://manager.geweapi.com/#/account/index)，点击去开通按钮，领取7天免费试用

    ![步骤1.png](https://api.apifox.cn/api/v1/projects/3475559/resources/430902/image-preview)
   
</Background>
:::
:::highlight purple 
**步骤2：** 兑换成功后，点击[Token中心](http://manager.geweapi.com/#/account/access)，生成Token，复制TokenId
<Background>
![步骤2.png](https://api.apifox.cn/api/v1/projects/3475559/resources/430901/image-preview)
    </Background>
:::

:::highlight purple 
**步骤3：** 打开[API文档](http://doc.geweapi.com/api-139908278)，携带TokenId参数即可进行登录模块代码接入，登录成功后，即可根据业务调用对应接口开发各类业务。
<Background>
![image.png](https://api.apifox.cn/api/v1/projects/3475559/resources/417540/image-preview)

<Card title="1. 如何发送消息‌" > 
在与微信交互中，开发者需先调用[获取通讯录列表接口](http://doc.geweapi.com/api-139908283)，获取好友/群的列表缓存入库，然后直接调用[发送消息相关接口](http://doc.geweapi.com/api-139908313)即可。
</Card>

<Card title="2. 如何接收消息" > 
在与微信交互中，用户可能会处理 好友/群 的消息接收，做到消息交互，可以[配置消息订阅](http://doc.geweapi.com/doc-3146208)，然后编写业务逻辑，在调用发送消息相关接口即可。)
</Card>
<Card title="3.如何开发群管理、自动化等操作" > 
市面上所有机器人操作，都是基于接收消息后的逻辑处理，例如群管理、消息保存、聚合聊天、消息托管、多群转发、内容直播、社区团购、消息转播、云发单、机器人自动回复等，所以开发者只需要[配置消息订阅](http://doc.geweapi.com/doc-3146208)，再加上业务逻辑即可自定义自己的机器人/客服系统
</Card>
<Card title="4.如何最快测试" > 
平台提供了[Apifox在线调试](http://doc.geweapi.com/api-139908278)，在网页界面就能运行测试，同时，平台也提供了[Demo演示](http://218.78.52.174:3456/#/login)，新用户未编写代码前可使用如上方式测试业务可行性。
</Card>
