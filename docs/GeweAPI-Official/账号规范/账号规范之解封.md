# 账号规范之解封

## 01 前提条件

> 协助解封的对方账号要满足以下条件，因此需要在养号期间，将此类账号添加为好友，以备不时之需。

1.  好友账号需绑定银行卡和手机号。
    
2.  好友账号注册时间需超过`6个月`，并且没有被封被冻结等情况；
    
3.  好友账号`1个月`内没有为其他账号做过辅助验证，`6个月`内没有为超过`2位`好友做过辅助验证，`1年`内没有为超过`3位`好友做过辅助验证。
    
4.  好友账号没有被其他账号邀请辅助解封。
    
5.  好友账号最近一年没有被封号过。
    

## 02 解封步骤

1.  好友搜索某信团队官方账号。
    
2.  进入某信团队官方账号，左下方点击自助工具。
    
3.  点击辅助验证 ➠ 协助好友验证 ➠ 确定是他本人联系我 ➠ 已经确认身份，出现已提交页面就可以了。
    
4.  按照提示依次输入验证信息。
    
5.  在解封的过程中，如果不小心错误输入帮你解封人的手机号，需要 `1 小时`以后再试。
    
6.  根据登录弹窗进入解封流程。
